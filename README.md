# Login System

Aplikasi sistem login & register sederhana menggunakan Node.js + Express.

## Fitur
- Register akun baru
- Login dengan JWT
- Edit profil
- Ganti password

## Cara Jalankan (Lokal)

```bash
npm install
node server.js
```

Buka browser: http://localhost:3000

## Environment Variables

| Variable | Keterangan |
|----------|-----------|
| PORT | Port server (default: 3000) |
| JWT_SECRET | Secret key untuk JWT |
